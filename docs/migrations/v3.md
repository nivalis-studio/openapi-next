# Migrating to v3

## Breaking changes

1. Fluent route builder APIs are removed.
2. `defineRoute(...)` is removed.
3. Routes now use `defineRouteContract(...)` + `bindContract(contract, handler)`.
4. OpenAPI generation reads contract files (`contract.ts` or `*.contract.ts`).
5. CLI can run with no required flags and resolves metadata from `package.json`.

## After (v3)

```ts
// src/app/api/users/contract.ts
import { defineRouteContract } from '@nivalis/openapi-next';
import { z } from 'zod';

export const getUsersContract = defineRouteContract({
  method: 'GET',
  operationId: 'listUsers',
  input: {
    query: z.object({ page: z.coerce.number().int().min(1) }),
  },
  responses: {
    200: {
      description: 'ok',
      content: {
        'application/json': {
          schema: z.object({
            success: z.literal(true),
            items: z.array(z.object({ id: z.string().uuid() })),
          }),
        },
      },
    },
  },
});
```

```ts
// src/app/api/users/route.ts
import { bindContract } from '@nivalis/openapi-next';
import { getUsersContract } from './contract';

export const GET = bindContract(
  getUsersContract,
  async (_request, _context, input) => ({
    status: 200,
    contentType: 'application/json',
    body: {
      success: true,
      items: await listUsers(input.query.page),
    },
  }),
);
```

## OpenAPI CLI

Minimal command:

```bash
openapi-next
```

Useful options:

```bash
openapi-next --app-dir src/app/api --output public/openapi.json
openapi-next --strict-missing-contracts
```
